### ✅ 2. [CSS] 흑(Black) 플레이어 기물 역방향 문제

**증상:**
- 흑 플레이어 시점에서 보드를 180도 회전시켰을 때 기물 이미지도 거꾸로 표시됨

**원인:**
- `.board-container.flipped #board-frame` 스타일이 보드 전체를 회전시킴
- 기물 이미지(`.piece-img`)에 대한 반대 회전 처리가 없었음

**수정 내용:**
```css
/* 보드가 회전할 때 기물은 반대로 회전하여 정방향 유지 */
.board-container.flipped .piece-img {
    transform: rotate(180deg);
}

/* 호버 효과도 회전 고려 */
.board-container.flipped .my-piece:hover .piece-img {
    transform: rotate(180deg) scale(1.1);
}
```

**수정 파일:**
- `src/main/resources/static/css/style.css` (라인 238-256)

**결과:**
- ✅ 흑 플레이어 시점에서도 기물이 정방향으로 표시됨
- ✅ 호버 효과도 정상 작동
